Définition des balises HTML5.

  • <!-- ... -->


    Un commentaire en HTML est un texte qui sert simplement de mémo. Il n'est pas affiché, il n'est pas lu par l'ordinateur, cela ne change rien à l'affichage de la page. Cela sert à vous et aux personnes qui liront le code source de votre page. Vous pouvez utiliser les commentaires pour laisser des indications sur le fonctionnement de votre page.

  • <a></a>


    L'élément HTML <a> (pour ancre ou anchor en anglais), avec son attribut href, crée un lien hypertexte vers des pages web, des fichiers, des adresses e-mail, des emplacements se trouvant dans la même page, ou tout ce qu'une URL peut adresser. Le contenu de chaque élément <a> doit indiquer la destination du lien.
    Exemples:
    Un lien vers un autre site
    <a href='https://example.com'>Site web</a>

    Un lien vers une adresse électronique (mailto:)
    <a href='mailto:m.blut@example.com'>Email</a>

    Un lien vers un numéro de téléphone (tel:)
    <a href='tel:+123456789'>Téléphone</a>

    Un lien vers une autre page de son site
    <a href='page2.html'>Page 2</a>

    Un lien vers une ancre (# vers id)
    <a href='#mon_ancre'>Aller vers l'ancre</a>
    <h2 id='mon_ancre'>Titre</h2>

    Lien vers une ancre située dans une autre page
    <a href='page2.html#rollers'>Rollers</a>
    ... vous amènera sur la page2.html, directement au niveau de l'ancre appelée rollers.

    Un lien qui affiche une infobulle au survol
    <a href='page2.html#rollers' title='Aller à la rubrique rollers'>Rubrique rollers</a>
    ...affichera une infobulle 'Aller à la rubrique rollers'

    Un lien qui ouvre une nouvelle fenêtre
    <a href='https://example.com' target='_blank'>Exemple</a>
    Lorsqu'un lien est doté d'un attribut target='_blank', un objet Javascript window.opener sera utilisable pour la page ouverte. Il donne la possibilité notamment de changer l'adresse de la page ouvrante (vers un site malveillant par exemple). Il est conseillé de rajouter l'attribut rel='noopener', l'objet window.opener sera alors inutilisable et défini à null dans la fenêtre ouverte. <a href='https://example.com' target='_blank' rel='noopener'>Exemple</a>

  • <address></address>


    Généralement utilisée dans le conteneur <footer>, cette balise est utilisée à la place de la balise <p>, les informations nécessaires pour contacter la ou les auteurs du document cité. Cela peut-être son nom ou un label suivi de son adresse email par-exemple ou adresse postale. Confusion possible : on utilisera la balise <p> pour donner l'adresse postale d'une entreprise sans relation avec l'auteur du document s'y rapportant.
    Exemple:
    <address>
    Ecrit par Jacques :

    </address>

  • <div></div>


    La balise <div> est un élément de bloc, qui spécifie les domaines ou les parties de document HTML. Il n’a aucun effet sur le contenu ou la mise en page et permet de regrouper des éléments HTML pour être stylisés avec CSS ou manipulés avec des scripts. La balise <div> est un élément de niveau bloc, donc un saut de ligne est placé avant et après. Vous pouvez placer n’importe quel élément HTML dans la balise <div>, y compris un autre <div>. La balise <div> ne peut pas être dans la balise <p>, car le paragraphe sera brisé au point où la balise <div> est saisi. Pour appliquer les styles à l'intérieur de paragraphe, utilisez la balise <span>, qui est utilisé avec les éléments en ligne.

  • <p></p>


    L'élément HTML <p> représente un paragraphe de texte. Les paragraphes sont généralement représentés comme des blocs et séparés par un espace vertical, leur première ligne est également parfois indentée. Les paragraphes sont des éléments blocs. Étant des éléments de bloc, les paragraphes se fermeront automatiquement si un autre élément de bloc est analysé avant la balise de fermeture </p>

  • <table></table>


    La balise <table> définit un tableau HTML.
    Chaque ligne du tableau est définie par une balise <tr>. Chaque en-tête de tableau est défini par une balise <th>. Chaque donnée/cellule du tableau est définie avec une balise <td>.
    Par défaut, le texte des balises <th> est en gras et centrée.
    Par défaut, le texte des balises <td> est régulier et alignée à gauche.
    Pour ajouter une légende à un tableau, utilisez la balise <caption>.
    Remarque : Les balises <td> sont les conteneurs de données de la table. Ils peuvent contenir toutes sortes d'éléments HTML ; texte, images, listes, autres tableaux, etc.
